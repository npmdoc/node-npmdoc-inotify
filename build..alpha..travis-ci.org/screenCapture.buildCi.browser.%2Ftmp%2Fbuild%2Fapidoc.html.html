```
You can call this function as many times as you want in order to monitor different paths.
**Monitoring of directories is not recursive**: to monitor subdirectories under a directory, additional *watches* must be created
.

  * `inotify.removeWatch(watch_descriptor)`: Remove a watch associated with the watch_descriptor param and returns `true` if the
 action was successful or `false` in the opposite case. Removing a watch causes an `Inotify.IN_IGNORED` event to be generated for
 this watch descriptor.

  * `inotify.close()`: Remove all the watches and close the inotify's file descriptor
. Returns `true` if the action was successful or false in the opposite case.
